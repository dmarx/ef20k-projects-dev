## ğŸ“‹ About This Repository

This repository catalogs community-developed tools for analyzing public records released by the House Oversight Committee on November 12, 2025. These documents, totaling over 20,000 pages, have been processed and made available through Hugging Face for research purposes.

**Our Purpose**: To provide researchers, journalists, and investigators with effective tools for navigating and analyzing these public records in a responsible manner.

> **Content Notice**: These documents contain sensitive material related to investigations of serious crimes. All tools listed here are designed for legitimate research, journalistic, and investigative purposes.
